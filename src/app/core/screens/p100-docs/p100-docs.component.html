Ultimate BFF: Plataforma de Gestión Avanzada para Documentación BFF en Proyectos

Ultimate BFF se presenta como una herramienta avanzada diseñada para facilitar la gestión de la documentación BFF en proyectos. Ofrece una plataforma completa que permite documentar de manera sistemática los endpoints genéricos, de pantalla y los componentes visuales asociados a cada proyecto, destacando por su enfoque en la organización y presentación estructurada de esta información.

Arquitectura y Funcionamiento de Ultimate BFF

La arquitectura de Ultimate BFF se basa en tres elementos esenciales:

  • Endpoints Genéricos: Funciones y operaciones centrales que no son accedidas directamente por el cliente.
  • Endpoints de Pantalla: Diseñados para atender a una pantalla específica del cliente, realizando llamadas a endpoints genéricos.
  • Componentes Visuales: Imágenes que representan los componentes de la aplicación, estableciendo una relación entre cada componente funcional y el endpoint al cual realiza la llamada.

Esta arquitectura simplifica la organización del código y permite realizar llamadas más específicas para los clientes, facilitando la reutilización de endpoints genéricos en diversas pantallas sin incurrir en prácticas indebidas.

Cómo Utilizar Ultimate BFF

  1. Dashboard

    2. Después de iniciar sesión, accede al dashboard que ofrece un menú lateral con información detallada del usuario. Permite acciones como cerrar sesión o crear un nuevo proyecto. En la sección derecha, se despliegan los proyectos a los cuales el usuario pertenece, proporcionando detalles clave como fecha de creación, nombre, cantidad de usuarios asociados, número de endpoints de pantalla y número de endpoints genéricos. Al hacer clic en un proyecto, serás redirigido a la pantalla correspondiente.

  2. Proyecto

    3. En la pantalla de un proyecto, encuentras cinco botones principales:

    • Participantes: Permite ver y gestionar los participantes del proyecto.
    • Nuevo Endpoint Genérico: Redirige al formulario para crear un nuevo endpoint genérico.
    • Nuevo Endpoint de Pantalla: Redirige al formulario para crear un nuevo endpoint de pantalla.
    • Nuevo Componente Visual: Redirige al formulario para crear nuevos componentes visuales.
    • Exportar a PDF: Genera un documento PDF organizado con la documentación BFF actual del proyecto.

    Debajo de estos botones, hay tres desplegables para visualizar los distintos endpoints genéricos, de pantalla y componentes visuales del proyecto.

  3. Formulario de Creación Endpoint Genérico

    4. Permite la creación de un nuevo endpoint genérico, donde debes especificar información como el método del endpoint, nombre, ruta de la petición, descripción, request params y request body si es necesario, además del objeto que devuelve el endpoint. También puedes especificar la consulta de base de datos necesaria para el funcionamiento del endpoint.

    La creación de objetos, request params, request body y response consta de un formulario "campo, valor" donde añades el nombre del campo y el tipo de dato del mismo. Este formulario puede variar desde un JSON simple hasta una estructura más compleja.

  4. Formulario de Creación Endpoint de Pantalla

    5. Permite la creación de un nuevo endpoint de pantalla, donde debes especificar información como el método del endpoint, nombre, ruta de la petición, enpoint genérico al que realiza la petición, descripción, request params y request body si es necesario, además del objeto que devuelve el endpoint.

    La creación de objetos, request params, request body y response consta de un formulario "campo, valor" donde añades el nombre del campo y el tipo de dato del mismo. Este formulario puede variar desde un JSON simple hasta una estructura más compleja.

  5. Formulario de Creación Componente Visual

    6. Permite la creación de un nuevo componente visual, donde debes especificar información como el nombre del componente o pantalla, distintas imágenes del componente con su descripción y una asociación entre las distintas partes del componente y el endpoint de pantalla al que realizan la petición. Puedes añadir tantas imágenes y llamadas como sea necesario.

  6. Visualización de Endpoints y Componentes

    7. Estas pantallas permiten la visualización ordenada de la información de los endpoints genéricos, de pantalla y componentes visuales. Cuentan con dos botones superiores, uno para eliminar el endpoint o componente y otro para su edición, que redirigirá al usuario al formulario citado anteriormente con los datos rellenos con la información del endpoint o componente deseado, permitiendo la edición del mismo.

    8. Al exportar nuestro BFF obtenemos un documento PDF con toda la información del proyecto ordenada, perfecto para entregar documentaciones.
